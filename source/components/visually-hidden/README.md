# VisuallyHidden

Use this to hide content visually, but not from screen readers.

### Author

[Pelle Bjerkestrand](mailto:pelle.bjerkestrand@creuna.no)

### Dependencies

- `prop-types`

### Usage

```javascript
<VisuallyHidden>Assistive text</VisuallyHidden>
```
